# IronPeak Construction Group

IronPeak Construction Group is a modern, full-service construction company delivering high-quality residential and small commercial projects across the Bay Area. Since 2014, we have established ourselves as a dependable builder that values structure, planning, and precise execution.

## 🚀 Features

- **Responsive Landing Page**: Built with modern web standards for a seamless experience across all devices.
- **Service Modules**: Detailed sections for Residential Construction, Home Remodeling, Commercial Build Outs, and more.
- **Dynamic UI**: interactive elements powered by Framer Motion for a premium look and feel.
- **Contact Integration**: Easy-to-use contact sections for project inquiries and free quotes.
- **Modern Performance**: Optimized with Next.js for fast load times and excellent SEO.

## 🛠 Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🏁 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm, yarn, pnpm, or bun

### Installation

1. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

2. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `src/app/`: Next.js App Router pages and layouts.
- `src/components/`: Reusable React components (Hero, About, Services, etc.).
- `public/`: Static assets like images and icons.
- `src/globals.css`: Global styles and Tailwind imports.
